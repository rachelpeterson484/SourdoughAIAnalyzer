# Sourdough Tracker

A comprehensive web application for tracking and improving your sourdough baking journey with AI-powered analysis and personalized recommendations.

## 🍞 Features

### **Recipe Management**
- Create and manage custom sourdough recipes
- Track ingredients, hydration levels, and flour combinations
- Calculate baker's percentages automatically
- Edit and refine recipes over time

### **Loaf Tracking**
- Log each bake with detailed parameters
- Upload exterior and crumb photos with smart compression
- Track fermentation times, temperatures, and proofing methods
- Rate your loaves and add personal notes
- Add inclusions (seeds, nuts, grains) with quantities

### **AI-Powered Analysis** 🤖
- **Computer Vision**: Analyze crumb structure from photos
- **Expert Recommendations**: Get personalized feedback on fermentation, technique, ingredients, and baking
- **Contextual Insights**: AI considers your notes, inclusions, and bake parameters
- **Actionable Steps**: Specific recommendations for your next bake
- **Free Integration**: Uses Hugging Face's free AI models with fallback analysis

### **Analytics Dashboard**
- Track baking progress over time
- Visualize ratings and improvements
- Compare different recipes and techniques
- Identify patterns in your baking

### **User Authentication**
- Secure Google Sign-In integration
- Personal data isolation per user
- Cross-device synchronization
- Profile management

## 🛠 Technology Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS
- **UI Components**: Radix UI, Lucide Icons
- **Backend**: Firebase (Firestore, Authentication, Storage)
- **AI/ML**: Hugging Face Vision Language Models
- **Image Processing**: Canvas API with smart compression
- **State Management**: React Hooks, Context API
